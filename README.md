# Meu Portfólio

Este é um portfólio simples criado com HTML e CSS.

## Estrutura do Projeto

- `index.html`: Arquivo principal do portfólio
- `styles.css`: Estilos CSS para o layout e aparência

## Como Visualizar

Abra o arquivo `index.html` em qualquer navegador web para visualizar o portfólio.

## Personalização

- Edite o conteúdo em `index.html` para adicionar suas informações pessoais, projetos e contatos.
- Modifique `styles.css` para alterar cores, fontes e layout conforme desejado.
